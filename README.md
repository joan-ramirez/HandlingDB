<h1>HandlingDB</h1>

## Documentation
Package for easy and fast manipulation of the database

```html
composer install joan-ramirez/handling-db
```

### The Aasiest Way To Export Data

```html
(new App\Joan-ramirez\Handling-DB)->export('C://ruta/',"users");
```
